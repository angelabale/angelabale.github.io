April 2023 - June 2023

Deutsche Telekom Service GmbH

Bonn, Germany

I was an intern in the Chapter Predictive Services and AI.

I analyzed the databases on the platform Hue and the codes of the project on CDSW to identify the origin of each token. I documented everything on an Excel sheet that contains the source table, the column(s) and the values of reference. In some cases, I wrote a description to explain how the token was generated and what it means. The file also contains the frequency of each token, with the frequencies lower than 100 highlighted so that the team can apply data reduction and enhance the model.

I wrote the docstring of each function in the project so that the team members and future members understand the codes easily.

I used the Python Black package to format each file of the project and make them PEP 8 compliant.
